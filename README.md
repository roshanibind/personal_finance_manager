# Personal Finance Manager

##Project Overview

A Python-based **Personal Finance Manager** that helps users track income, expenses, and savings from the command line. The application is designed to be simple, lightweight, and easy to extend.

##Features

* Add and categorize income
* Add and categorize expenses
* View balance summary
* Track spending by category
* Simple CLI-based interaction

## Tech Stack

* **Language:** Python 3.x
* **Libraries:** Listed in `requirements.txt`

## Setup Instructions

Follow these steps to run the project locally:

###Clone the Repository

```bash
git clone https://github.com/roshanibind/personal_finance_manager.git
cd personal_finance_manager
```

###Create a Virtual Environment (Optional but Recommended)

```bash
python -m venv venv
source venv/bin/activate      # On Linux/Mac
venv\Scripts\activate         # On Windows
```

###Install Dependencies

```bash
pip install -r requirements.txt
```

###Run the Application

```bash
python main.py
```
